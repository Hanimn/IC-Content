# Hint 1

The Orion Monitor login page is asking for credentials. You don't have them — but that doesn't mean no one does.

Systems like this are often deployed quickly, with placeholder credentials set during installation and never changed. Think about the most common username/password combinations used for internal admin panels. Try a few.

What's the most obvious thing someone might use as a password when they're just "setting things up for now"?
