# Module 08: The Lock With No Key
## The most common way in isn't a hack — it's a guess.

| | |
|---|---|
| **Difficulty** | Beginner |
| **Time** | ~35 min reading + ~30 min challenge |
| **Prerequisites** | [Module 01 — View Source](../01-view-source/README.md) |
| **Tools** | Web browser (Chrome or Firefox) |

---

## What You'll Learn

- Recognise login forms that may use default or hardcoded credentials
- Find credentials stored in client-side JavaScript using View Source
- Understand why hardcoded and default credentials are a critical vulnerability

## Quick Start

1. Read the [Ethical Hacking Pledge](../../ETHICAL-HACKING-PLEDGE.md) if you haven't yet
2. Open `lesson.md` and read it top to bottom
3. Open `challenge/index.html` in your browser
4. Find the flag — it looks like `FLAG{...}`

**Stuck?** Wait 10 minutes before opening hints. Struggling is part of learning.
