# Module 07: The Form's Secret Life
## Forms carry more data than they show.

| | |
|---|---|
| **Difficulty** | Beginner |
| **Time** | ~30 min reading + ~25 min challenge |
| **Prerequisites** | [Module 02 — Browser Inspector](../02-browser-inspector/README.md) |
| **Tools** | Web browser (Chrome or Firefox) |

---

## What You'll Learn

- Find hidden `<input type="hidden">` fields in a form using DevTools
- Edit hidden field values inline in the Inspector and submit the modified form
- Understand why hidden fields are not a security boundary

## Quick Start

1. Read the [Ethical Hacking Pledge](../../ETHICAL-HACKING-PLEDGE.md) if you haven't yet
2. Open `lesson.md` and read it top to bottom
3. Open `challenge/index.html` in your browser
