# Hint 2

Use **View Source** to see the raw HTML your browser received:

- **Windows / Linux:** `Ctrl+U`
- **Mac:** `Cmd+Option+U`
- **Any OS:** Right-click anywhere on the page → "View Page Source"

A new tab will open showing the raw HTML. Once you're there, press `Ctrl+F` (or `Cmd+F` on Mac) and search for the word `FLAG`.

While you're in there, look for HTML comment syntax too — comments start with `<!--`. Developers sometimes leave notes to themselves that they forget to remove.
