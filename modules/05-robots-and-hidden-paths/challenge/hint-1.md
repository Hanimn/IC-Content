# Hint 1

The main page of the Meridian Publishing Group site looks like a normal news website. Nothing obviously out of place.

But every website has a file that acts like a map of its contents — a file designed to tell search engines which parts of the site to index, and which to skip. That file is publicly accessible to anyone who knows where to look.

What would a site owner put in that file if there were paths they wanted to keep out of search results?
