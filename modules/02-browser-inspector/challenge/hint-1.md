# Hint 1

There's content on this page that isn't shown on screen — but it's definitely in the DOM.

In Module 01 you used View Source to find something hidden in the raw HTML. This module's technique goes further: the Inspector shows you the *live* structure of the page, element by element, including elements that CSS has made invisible.

Open DevTools (F12) and look through the Elements panel. Not everything visible on screen is everything that exists. The flag is in two pieces — think about the different ways an element can be present in the DOM but invisible to a visitor.
