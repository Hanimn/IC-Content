# Module 03: Digital Breadcrumbs
## Every visit leaves a trace — and that trace can be changed.

| | |
|---|---|
| **Difficulty** | Beginner |
| **Time** | ~30 min reading + ~25 min challenge |
| **Prerequisites** | [Module 02 — Browser Inspector](../02-browser-inspector/README.md) |
| **Tools** | Web browser (Chrome or Firefox) |

---

## What You'll Learn

- Read cookies stored by a webpage using the DevTools Application tab
- Modify a cookie value to change your apparent membership tier and unlock hidden content
- Decode a base64-encoded cookie to reveal data hidden inside it

## Quick Start

1. Read the [Ethical Hacking Pledge](../../ETHICAL-HACKING-PLEDGE.md) if you haven't yet
2. Open `lesson.md` and read it top to bottom
3. Open `challenge/index.html` in your browser
4. Find the flag — it looks like `FLAG{...}`

**Stuck?** Wait 10 minutes before opening hints. Struggling is part of learning.
