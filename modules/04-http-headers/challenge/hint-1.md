# Hint 1

The page is making more than one HTTP request. When a browser loads a webpage, it can fetch additional resources in the background — scripts, images, data files — each as a separate request.

Open the Network tab in DevTools (`F12` → **Network**) and then **reload the page**. The Network tab only records requests made while it's open.

After reloading: how many requests do you see in the list? Are they all for the main page?
