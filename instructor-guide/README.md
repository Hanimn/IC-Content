## Quick Start for Instructors

1. Read the [Ethical Hacking Pledge](../ETHICAL-HACKING-PLEDGE.md) — have students read and sign it before any module
2. Work through Module 00 yourself before teaching it — the orientation module establishes expectations for the whole series
3. Change the flag values before assigning challenges to students (see [deployment-guide.md](deployment-guide.md))
4. Review [classroom-setup.md](classroom-setup.md) for pacing and assessment guidance

## Series Philosophy

This series teaches offense to build defense intuition. Every technique students learn represents a real class of vulnerability. The goal is not to produce hackers — it's to produce developers and security thinkers who understand why certain coding patterns are dangerous.

For ages 11–14, the ethical framing matters as much as the technical content. The pledge isn't bureaucratic overhead — it's the foundation that makes the rest safe to teach.
