# F08 — ASCII All the Way Down

> Every letter becomes a number. And groups of numbers can become text that looks like nonsense — until you know the trick.

---

## Module Metadata

| Field | Details |
|---|---|
| **Difficulty** | Beginner |
| **Time** | ~30 min reading + ~25 min challenge |
| **Prerequisites** | F05 (Everything Is a Number), F06 (Thinking in Hex) |
| **Tools** | Web browser |

---

## What You'll Learn

- Use the ASCII table to decode sequences of decimal numbers into readable text
- Recognise base64-encoded strings by their distinctive appearance
- Decode base64 using a built-in decoder and identify when a character is wrong
