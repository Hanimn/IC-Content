# Hint 2 — What you're looking for

If you've clicked all five files and you're still not sure what you're seeing, here's what to look for.

Four of the five files will show two hashes that look **identical** — same first character, same last character, same everything in between. The page will mark these green and tell you the file is genuine.

**One file will show two hashes that look like completely unrelated random strings.** Not a few characters different — totally different. That's the avalanche effect from the lesson, and it means the file you have is not the file the publisher built.

Look at the filename of that one mismatched file. Type it into the submission box exactly as it appears (including dashes, no extra spaces). Then click Submit.

Hint within a hint: of the five files, which one is the most attractive target for an attacker who wants to slip malicious code onto a victim's computer? The release notes? The README? Or one of the executables?
