F14 — THE ONE-WAY DOOR

Encryption is a door with a key. Hashing is a door with no door.
BRIEFING. The publisher of OrionScan has just released version 1.0.0. They've posted the official SHA-256 hash of every file in the release on their website. You've downloaded the package — but you've heard rumors that an attacker may have replaced one of the files with a tampered copy on a mirror server. Your job: hash each file, compare to the publisher's official hashes, and identify which file has been tampered with.
