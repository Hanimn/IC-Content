Welcome to Layered Labs

Every webpage is three languages stacked. Read all three layers to assemble your access flag.
One language alone tells you only a third of the story.

Layer 01

HTML // the bones

HTML defines what's on the page. Tags like <p>, <h1>, and <button> wrap your content and give it structure. The browser uses HTML to know what elements exist and how they nest. Comments wrapped in <!-- --> are skipped when rendering, but they are still in the source.

Layer 02

CSS // the paint

CSS controls how the page looks. Selectors target elements; properties set their colour, size, position, and visibility. CSS can hide content with display: none — and it can also add content using content: on ::before and ::after pseudo-elements.

Layer 03

JavaScript // the brain

JavaScript makes the page do things. Variables hold values; functions do work; event listeners react to clicks and key presses. JS can change the page after it has loaded — which means what you see on screen and what View Source shows can sometimes differ.
