# Hint 1

**Panel 1** (Binary bit-toggle):
You need to make the bits add up to 42.
The place values are 128, 64, 32, 16, 8, 4, 2, 1.
Which of those add up to 42? Start with the largest one that fits.

**Panel 2** (Binary → Hex):
Split the 8-bit number into two groups of 4 bits.
Each group of 4 bits corresponds to exactly one hex digit.
The table on the page shows you which 4-bit pattern maps to which hex digit.

**Panel 3** (Hex → ASCII):
Look up each byte in the ASCII table on the page.
The first byte is `46`. Find `46` in the table — what character does it map to?
Work through them one by one.
