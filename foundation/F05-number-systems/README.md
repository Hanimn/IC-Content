# F05: Everything Is a Number

> Computers don't know what "A" is. They only know 65. And 65 is really 01000001.

---

## Module Info

| Field         | Details                                           |
|---------------|---------------------------------------------------|
| Difficulty    | Beginner                                          |
| Time          | ~30 min reading + ~25 min challenge               |
| Prerequisites | None (can be taken alongside F01 independently)   |
| Tools         | Web browser                                       |

---

## What You'll Learn

- Convert numbers between decimal, binary, and hexadecimal
- Explain why computers use binary instead of decimal
- Decode ASCII text from a sequence of hex bytes

## Quick Start

1. Read the [Ethical Hacking Pledge](../../ETHICAL-HACKING-PLEDGE.md) if you haven't yet.
2. Read `lesson.md` from top to bottom (~30 min)
3. Keep `concept-cards.md` open as a reference
4. Open `challenge/index.html` in your web browser
5. Work through all three panels to decode the flag
6. Check `challenge/hint-1.md` if you get stuck
