# Hint 1 — F16

(Try the challenge with just `help` and `ls` first before reading hints. Curiosity beats answers.)

---

The Discoveries panel on the right shows three pips. Each one corresponds to a different hiding method. Each method requires a different command:

- **Pip 1** — find readable text inside something that *isn't* readable. The terminal even tells you which command does this when you try the wrong one.
- **Pip 2** — at least one file is lying about what it is. The lesson said the filename can lie. Which command doesn't trust the filename?
- **Pip 3** — two files share a word in their names. You don't have to know which two — there's a command that *finds* files by name.

If you've run `ls` once, type `help` to see the full list of commands. Then think about which command answers each of the three questions above.

You will not get there by guessing. Run real commands.
