## What You'll Learn

- What a computer actually is, underneath the screen and the apps
- The difference between programs, files, folders, and paths
- What a "browser" is, what a "server" is, and how they're related
- The difference between source code and a running program
- The single ethics rule that the rest of this series depends on
- A small set of keyboard shortcuts you'll use in every later module

## Quick Start

1. Read the [Ethical Hacking Pledge](../../ETHICAL-HACKING-PLEDGE.md) — short, signable, and the rule the whole curriculum is built on.
2. Open `lesson.md` and read through, then open `challenge/index.html` in your browser.

You don't need to install anything. You don't need to know any code. If you've ever clicked an icon on a screen, you have enough background to start.
