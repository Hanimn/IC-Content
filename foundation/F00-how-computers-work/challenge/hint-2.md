# Hint 2

Direct answers per zone, in case you are stuck:

1. **The File Browser.** Paths start at the root and walk down. The right answer begins with `/Users` and ends with `cat.jpg`.

2. **The Long File.** Press **Ctrl+F** (Windows/Linux) or **Cmd+F** (Mac) to open the browser's Find bar. Type `IMPORTANT_CODE`. The browser will jump to the line containing that text. The value is a Greek letter — and it's the simplest one.

3. **The Folder Tree.** In `/home/projects/secret/key.txt`, the folder named just before `secret` is the one that contains it.

4. **The Browser Window.** Chrome, Safari, and Firefox are all the *same kind of thing* (programs that show webpages). They are not the operating system, not the internet, and they are not fakes of each other.

5. **The Server.** Re-read the lesson: a server is regular hardware, not a special device. It is defined by what it does, not what it is.

6. **The Ethics Panel.** Only one answer respects the rule "just because you can, does not mean you should."

If you've solved all six zones and the boot-up button still hasn't appeared, refresh the page and try again. Your progress is per-session.
