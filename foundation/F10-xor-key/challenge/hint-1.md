# Hint 1

XOR is all about the bits. Level 1 is asking you to fill in the truth table — the rule is simple: **the output is 1 when the two inputs are different**. Zero and zero are the same, so output is 0. Zero and one are different, so output is 1. Apply that rule to each row.

For Level 2, work through the bytes one column at a time. Each column is just one XOR operation using the rule from Level 1.

For Level 3, the clue on the page tells you the key's decimal value. What ASCII character has a decimal value of 33? Check back at your notes from F08 (ASCII and Encoding) if you need a refresher.
