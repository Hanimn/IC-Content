## Question 1: HTTP Method in Exchange 2

The method starts with **P** and rhymes with "ghost."

---

## Question 2: Status Code in Exchange 3

The status code starts with **4**. In HTTP, codes starting with 4 mean the client did something wrong — or doesn't have permission to access something.

---

## Question 3: The Complete Flag

- `X-Fragment-1` is in the **Exchange 2 response** headers
- `X-Fragment-2` is in the **Exchange 3 response** headers

Put them together in order (Fragment 1 first, then Fragment 2). No spaces between them — they join directly.
