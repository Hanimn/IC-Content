# Hint 1

The simulation is asking you to trace the path a domain name takes to become an IP address.

Think about the order of steps: before your browser can visit a website, it needs to know *where* to go. That "where" is an IP address. But you typed a name, not a number.

Something has to convert that name into a number. That something is called DNS.

Work through the questions from the beginning — each one follows logically from the last.
