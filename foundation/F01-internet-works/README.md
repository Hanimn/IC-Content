# F01: The Postman Problem

> The internet is just billions of letters being delivered to the right address.

## What You'll Learn

- Trace how a domain name becomes an IP address through DNS lookup
- Understand why web traffic travels in packets instead of whole files
- Recognise the difference between a domain name and an IP address
