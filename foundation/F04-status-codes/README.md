## What You'll Learn

- Identify HTTP status code groups (2xx, 3xx, 4xx, 5xx) and what each range means
- Recognise the 6 most important status codes: **200, 301, 302, 403, 404, 500**
- Use View Source (Ctrl+U) to find content hidden in a page's HTML comments
