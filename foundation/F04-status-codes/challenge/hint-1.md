# F04 Challenge — Hint 1

The switchboard has six incoming situations — each one ends with a server giving a three-digit response.

Think about what each **group** means before you try to match the specific code:

- **2xx:** things went well — the request succeeded
- **3xx:** the server is pointing you somewhere else — follow the arrows
- **4xx:** something went wrong on your side, or you're not allowed
- **5xx:** something went wrong on the server's side — the server broke

**Strategy:** Match the tone of each scenario to the right group first, then pick the specific code within that group.

- Which scenarios sound like success?
- Which ones involve moving to a different location?
- Which ones involve being blocked or something missing?
- Which one involves a crash?

Once you've sorted the six situations into their groups, the specific code for each one will be much easier to identify.
