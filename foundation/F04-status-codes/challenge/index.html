AGENCY TRAINING // THE STATUS CODE SWITCHBOARD

Connect each situation to the correct HTTP status code
1xx = Informational 2xx = Success 3xx = Redirect 4xx = Client error 5xx = Server error

For each scenario below, select the correct HTTP status code. Then press CHECK ALL.

SITUATION 01
You typed a URL and the page loaded perfectly. The server sent back the HTML you asked for.
SITUATION 02
This website moved to a new domain six months ago. Any link to the old domain will work forever — but you'll always end up at the new address.
SITUATION 03
You just submitted a login form. The server is sending you to the dashboard — but only until your session expires.
SITUATION 04
You found a path called /admin. The server knows you're there, but you're not allowed in.
SITUATION 05
You followed a link from an old forum post, but the page it pointed to was deleted years ago.
SITUATION 06
You sent an unusually long string to a web form. The server couldn't handle it and crashed.
