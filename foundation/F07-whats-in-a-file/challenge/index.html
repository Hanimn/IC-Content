The first few bytes of any file reveal its true format. Extensions can lie. Bytes cannot.

How to Solve
For each file card, look at the hex bytes shown. Find those opening bytes in the reference table above. Select the matching format from the dropdown.
Some formats have ASCII-readable signatures — look at the ASCII column. %PDF, PK, GIF8, and MZ are all recognizable as text. Others (like JPEG's FF D8 FF) are non-printable.
After matching all six correctly, the bonus mission unlocks. Use the reference table to find the correct first byte for a PNG file.
