# F06 — Thinking in Hex

> Two digits in hex can represent anything a byte can hold. That's why everyone in security speaks it.

## What You'll Learn

- Read a hex dump in the classic offset / bytes / ASCII layout used by every hex editor
- Identify printable ASCII characters in raw hex data by their byte range (0x20–0x7E)
- Explain why security tools display data in hex rather than decimal or binary
