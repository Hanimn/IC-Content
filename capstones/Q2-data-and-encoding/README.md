Tier 2 · Data & Encoding · gates after F05–F08

---

When this capstone is built, it will require you to combine the four Tier 2 skills:

- **F05** — binary, decimal, hex
- **F06** — hex dumps
- **F07** — file magic bytes
- **F08** — ASCII and Base64

The challenge: a corrupted file with the wrong magic byte. Spot the corruption in a hex dump, identify what file type it claims vs. what it is, fix the byte, then decode an ASCII or Base64 payload hidden inside.

For now, [Q1 — The Mystery Letter](../Q1-fundamentals/) is the live capstone. Come back here after the others land.
