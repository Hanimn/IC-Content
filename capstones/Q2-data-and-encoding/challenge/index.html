Capstone Q2 — Repair the Artifact

Tier 2 · Data & Encoding · gates after F05–F08

Coming soon.

This capstone will require you to combine the four skills you built in Tier 2 (Data & Encoding):

  • F05 — binary, decimal, hex conversions
  • F06 — reading hex dumps
  • F07 — file magic bytes and what makes a PNG a PNG
  • F08 — ASCII tables and Base64

The challenge: a corrupted file with the wrong magic byte. You'll need to spot the corruption in a hex dump (F06), figure out what file type it claims to be vs. what it actually is (F07), apply the fix (F06 + F05), then decode an ASCII or Base64 payload hidden inside (F08).

For now, the lesson tier is here but the chained challenge isn't built yet. Come back after the other capstones land — Q1 (Fundamentals) is fully playable in the meantime.
