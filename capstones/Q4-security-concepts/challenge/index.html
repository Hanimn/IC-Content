Capstone Q4 — The Final Drive

Tier 4 · Security Concepts · gates after F12–F16

Coming soon.

This is the foundation track's graduation capstone. It will require you to chain the five skills you built in Tier 4 (Security Concepts):

  • F12 — memory and the stack
  • F13 — HTML, CSS, and JavaScript layers
  • F14 — hashing and file integrity
  • F15 — EXIF metadata and steganography
  • F16 — the unix shell toolbox

The challenge: investigate a leaked drive. A webpage references a file by its SHA-256 hash; one of the files on the drive matches; that file is a steganographically-modified photo whose EXIF GPS points to an offset in a binary; the binary has the flag at that offset. Five steps. All five tools.

For now, the tier is here but the chained challenge isn't built yet. Come back after the other capstones land — Q1 (Fundamentals) is fully playable in the meantime.
