Tier 4 · Security Concepts · gates after F12–F16

---

When this capstone is built, it will be the foundation track's graduation challenge. Combine all five Tier 4 skills:

- **F12** — memory and the stack
- **F13** — HTML, CSS, and JavaScript layers
- **F14** — hashing and file integrity
- **F15** — EXIF metadata and steganography
- **F16** — the unix shell toolbox

The challenge: investigate a leaked drive. A webpage references a file by SHA-256 hash; one of the drive's files matches; that file is a steganographically-modified photo whose EXIF GPS points to an offset in a binary; the binary has the flag at that offset. Five steps. All five tools.

For now, [Q1 — The Mystery Letter](../Q1-fundamentals/) is the live capstone. Come back here after the others land.
