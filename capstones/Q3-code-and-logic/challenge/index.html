Capstone Q3 — Extract the Embedded Secret

Tier 3 · Code & Logic · gates after F09–F11

Coming soon.

This capstone will require you to combine the three skills you built in Tier 3 (Code & Logic):

  • F09 — Caesar cipher / ROT13
  • F10 — XOR encryption with a repeating key
  • F11 — extracting strings from a compiled binary

The challenge: a fake compiled program contains an XOR-encrypted payload. The decryption key is hidden in the program's strings — but it's been Caesar-shifted to throw off a casual reader. You'll strings the binary (F11), recognize the shifted hint, brute-force or eyeball the Caesar shift (F09), then XOR-decrypt the payload to recover the flag (F10).

For now, the lesson tier is here but the chained challenge isn't built yet. Come back after the other capstones land — Q1 (Fundamentals) is fully playable in the meantime.
